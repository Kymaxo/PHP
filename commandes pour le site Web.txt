Dans le dossier Racine du projet

- Créer un fichier .yarnrc
- Ajouter la ligne suivante dans le fichier .yarnrc
(" --modules-folder www/yarn-assets")
- Depuis le dossier racine, lancer la commande: yarn install
- Supprimer le dossier ("node_modules")
- Ajouter le texte suivant: dans .gitignore
("/www/yarn-assets)
- créer un dossier controleur
- copier tous les fichiers dans le dossier controleur

- copier le fichier "bonjour.html" dans le dossier vues
